# Jiya-Daulat-Task-2
In this task, I analyzed a suspicious email to identify phishing characteristics and understand how attackers try to trick users.

# Phishing Email Summary
# Sample Email

I used a sample email screenshot which claims to be sent by Donald Trump and asks for filling a grant application.

# Analysis

# 1. Sender Email

The sender email shows `therealdonaldtrump@whitehouse.gov`. This looks suspicious because such high profile official emails are rarely used in this way and can be spoofed.

# 2. Header Analysis

Full email headers were not available in the screenshot. However, header analysis can be done using tools like MXToolbox to check SPF, DKIM, and sender authenticity.

# 3. Suspicious Link

The email contains a link “Grant Application” which may redirect to a fake website to collect sensitive information.

# 4. Social Engineering

The email uses a famous personality’s name to gain trust and manipulate the user.

# 5. Unusual Content

The message content does not match realistic communication from such a sender, making it suspicious.

# 6. Grammar and Language

The email uses informal phrases like “very very good work” and “bigly successful” which is not professional.

# 7. Lack of Verification

There is no proper signature, contact details, or verification method in the email.

# Phishing Summary

* Possible email spoofing
* Suspicious link
* Social engineering tactics
* Unprofessional language
* Lack of proper authentication

# Conclusion

The email contains multiple phishing indicators and is likely a phishing attempt.
